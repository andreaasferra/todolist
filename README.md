TO DO List App – Flutter
Descrizione

Questa applicazione è una TO DO List sviluppata in Flutter.
Permette all’utente di gestire una lista di attività, segnare quelle completate e visualizzare alcune statistiche sull’andamento delle task.

L’app è composta da due schermate principali: una per la gestione della lista e una per le statistiche.

Funzionalità
Schermata "Lista":

Visualizza l’elenco delle task inserite;

Aggiunta di nuove task;

Possibilità di segnare una task come completata.

Visualizzazione immediata dello stato delle attività.

Schermata "Stats".

Numero totale di task inserite.

Numero di task completate.

Numero di task ancora da svolgere.

Percentuale di efficienza (task completate / task totali)

Funzionalità opzionale

L’app può essere estesa per supportare più liste:

Creazione di più liste con nomi personalizzati (es. Spesa, Scuola, Lavoro).

Ogni lista ha le proprie task.

La schermata Stats mostra statistiche separate per ogni lista.
